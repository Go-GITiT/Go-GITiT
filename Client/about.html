GoGitit

Data Visualization of Popular JavaScript Front End Frameworks

GoGitit is a node project that leverages the Google BigQuery API and Github's API to gather data about the usage statistics of different front-end frameworks

The Problem

GitHub has no means to provide information about what frameworks are being used. The goal of this project is to gather and parse GitHub repository information to create accurate usage metrics of JavaScript frontend frameworks.

Solution

To achieve this goal we used Google BigQuery to retrieve daily activity from GitHub for all javascript repos. We would then use GitHub's API to fetch files to use as framework identifiers. By parsing through the package.json and index.html files of each repo we keep individual records of the frameworks being used and keep a cumulative count of the occurences to display using a series of D3 data visualization graphs. Support for more frameworks will be added over time.

The Team

GoGITiT is a group project by Marc Christophe, Doris Hernandez, Demesheo Wilson, and Ryan McBride at Telegraph Academy.
